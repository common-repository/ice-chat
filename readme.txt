== Description ==

ICE Chat is a web-based customer interaction software designed to provide quality live chat interactions on the client’s websites. ICE Chat offer a significant value proposition to businesses in terms of cost saving, increased customer satisfaction, and generating higher online revenues. Businesses can leverage Chat to escalate their communication efforts and convert visitors into customers.

= Special Features =

Browser Based Application : Use any browser to login from anywhere and you are ready to answer chats.

SSL Secure Chat : All chat messages are transmitted using 256-bit encryption, providing a secure connection.

Automatic Chat Distribution : Visitors are automatically directed to the first available agent.

Customized Chat Window and Button : Select and customize the button from gallery to match your brand.

Online Conversion Tracking : We help you set up your online conversion tracking so you will know the ROI impact of your chat efforts.

Analysis and Reporting : Measure the performance of your staff for each interaction and visualize the real time impact of chat on your KPI’s.

Multiple Security Roles : You can assign permissions based on different roles you create to various features, reports and administrative components.

Canned Responses : With one click delivery, agents can send pre-written responses to commonly asked questions.

Chat Transcripts : All Chat interactions are stored in our database for quality review and consistency training.

Post Chat Survey : ICE Chat supports further quality analysis of every chat interaction and helps you increase customer satisfaction.

Geographical location : Tracks the visitor’s location, IP address, browser details.

Typing Indicator : Allows visitors and staff to know that other side is busy composing a reply.

Proactive Chat : Triggers chat invitations automatically based on pre-set rules.


== Installation ==

1. Upload the ICE Chat plugin to `/wp-content/plugins/plugin-name` directory Wordpress Plugin or it can be installed through WordPress screen directly. 
2. Activate the plugin by “Plugins” screen
3. Use Chat Box Screen for plugin configuration.

==Frequently Asked Questions==

Q. What is a chat button?
A. A chat button helps a visitor to connect with an agent or supervisor. 

Q. Can a customer chat with agent and supervisor at the same time?
A. Yes, a customer can chat with agent and supervisor at same time. However, it depends upon the access provided by organization. 

Q. How are the chats exchanged between customer and agents/supervisors?
A. Exchange of chats or messages are possible through the medium of chat interface.
